GoIT JavaScript Homework 01

Завдання 1: Замовлення дроїдів

Відкрийте консоль розробника (F12) щоб побачити результати виконання завдань.

Завдання 2: Доставка товару

Результати розрахунків доставки виводяться в консоль.

Завдання 3: Ширина елемента

Розрахунки ширини CSS елементів доступні в консолі.

Щоб побачити результати:
1. Натисніть F12 або правою кнопкою миші → "Inspect Element"
2. Перейдіть на вкладку "Console"
3. Подивіться результати виконання всіх завдань
